# Runstreet Sneaker Store

Runstreet je React + Vite web aplikacija za online prodaju patika. Projekat je prebacen sa beauty shop teme na sneaker shop iskustvo: katalog modela, detalji proizvoda, korpa, registracija/prijava i admin panel za upravljanje ponudom.

## Tehnologije

- React
- Vite
- React Router DOM
- Tailwind CSS
- json-server

## Pokretanje

```bash
npm install
npm run server
npm run dev
```

Frontend radi na `http://localhost:5173`, a fake API na `http://localhost:5000`.

## Funkcionalnosti

- pregled sneaker kataloga
- detaljna stranica svakog modela
- lokalna korpa
- registracija i prijava korisnika
- admin panel za dodavanje, izmenu i brisanje modela
